# מילון מושגים ותרגול — Industry 4.0

כלי לימוד אישי לקורס **"Industry 4.0 — יזמים ומנכ״לים משתפים"**
לימודי תואר שני, אוניברסיטת בר־אילן · מרצה: תא״ל (מיל׳) שגיב שרביט

## מה זה
אפליקציית ווב בקובץ יחיד (`מילון-מושגים-Industry40.html`), הפועלת בדפדפן **ללא התקנה, ללא שרת וללא אינטרנט**. מיועדת לשמש כחומר עזר במבחן (חומר פתוח), ומוגשת במסגרת מטלת הבונוס של הקורס.

## תכונות
- **מילון מושגים** — כל מונח עם הגדרה תמציתית, הסבר מורחב, דוגמה יישומית ושיוך לשבוע ולנושא. חיפוש וסינון מיידיים.
- **מאגר שאלות תרגול** — שאלות רב־ברירה במצב תרגול (משוב מיידי) או מבחן (ציון מסכם). טעות מפנה ישירות למושג במילון.
- **בקרת שאלות** — רק שאלות שאושרו על ידי המרצה נכללות בתרגול/מבחן; שאלה חדשה נוספת כ"ממתינה לאישור".
- **ניהול** — הוספת מושגים ושאלות, אישור שאלות, וייצוא/ייבוא הנתונים (JSON) לגיבוי.
- **הדפסה** — הפקת גיליון מילון נקי להדפסה, לשימוש במבחן אם לא מאושר שימוש במחשב.

## שימוש
פתחו את `מילון-מושגים-Industry40.html` בדפדפן (לחיצה כפולה). הנתונים נשמרים מקומית בדפדפן; לגיבוי — "ניהול → ייצוא JSON".
